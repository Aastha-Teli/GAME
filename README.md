# Rock Paper Scissor Game
A simple Rock Paper Scissor game built with Python.
## Features
* User vs Computer gameplay
* Random computer choice using Python's `random` module
* Win, Lose, and Tie detection
* Play Again option
* Beginner-friendly Python project

## How to Run
1. Clone the repository: git clone https://github.com/Aastha-Teli/GAME.git
2. Navigate to the project folder: cd GAME
3. Run the Python file: python paper.py
